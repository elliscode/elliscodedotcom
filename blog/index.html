How to send Twilio messages using AWS Lambda

If you are already registered with Twilio, dealing with a whole new registration process for Amazon SNS might not be worth the trouble, or may not meet your needs. The good news is it's very easy to get started using Twilio on AWS Lambda, and if you set it up right, you will be able to send messages from any AWS service that you please. Follow along with this guide and you will be up and running in no time!

If you want to jump right to the code, you can visit the Github repo here

This guide will be set up in three sections:

  • Creating your lambda
  • Creating your SQS queue to feed your lambda
  • Add a message to your SQS Queue from another AWS service (we will use both an EC2 instance and a lambda function)
